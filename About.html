ABOUT THE GAME

The Purpose of this game

This game was built as a class project CSC414-Software Engineering 1 at the University of Southern Mississippi.  Our project learn how to successfully create software while working in a team. This project is a game built in three.js designed to run in a web browser.   Our team consisted of five people.

Building the game

  • The game was built using ThreeJS/WebGL
  • Models were found on blenderswap and modified using blender to work with ThreeJS
  • Github was used for version control
  • Google+ was used for team communication outside of meeting times
  • Special Thanks to Jacob Bovatsek for LOADER.js to texture/model caching and for Javascript class design
